# cnweb-betmgm-hub

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态网页形式存在，不针对任何特定域名或具体网站，仅作为通用前端资源存放与展示用途。

仓库名称中的 “cnweb” 表示该仓库主要面向中文网页资源，“betmgm” 为内部项目标识，“hub” 表示其作为页面集合中心的定位。

## 目录说明

```
.
├── README.md          # 本说明文件
├── index.html         # 仓库主页（可选，若存在则用于页面导航）
├── pages/             # 存放所有独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 公共资源目录（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- **pages/**：每个 HTML 文件对应一个独立的页面，彼此之间无依赖关系。
- **assets/**：存放所有页面共用的样式、脚本和图片资源，如需单独引用可自行调整路径。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容保持中立，不包含任何具体商业推广或诱导点击元素。
- 归档的页面可能包含演示性内容、工具页面或通用模板，其设计目的仅为展示前端功能或布局。
- 每个页面均可在浏览器中直接打开运行，无需额外配置。

## 维护说明

- 本仓库由维护者定期更新，添加新的页面或修正现有页面的问题。
- 如果发现页面无法正常显示或存在兼容性问题，欢迎提交 Issue 反馈。
- 不接受针对特定域名或商业用途的定制请求，本仓库仅用于通用归档。
- 所有页面均遵循开源精神，可自由查看、复制和修改，但使用者需自行承担相关责任。
